# SubCompare

Flutter uygulaması: Abonelikleri puanlayarak karşılaştır, tablo ve yıldız görünümü ile öneri ver.

## Kurulum
```
flutter pub get
flutter run
```
Android Studio: `Open > subcompare`

## Özellikler
- Servis ekle/düzenle/sil
- 6 kriterde (Model, Görsel, Video, Entegrasyon, Gizlilik, Değer) 0..5 skor
- Erdinç ön ayarı (Görsel+Video+Android ağırlıklı)
- Genel skor ve öneri
- Kalıcı yerel kayıt (SharedPreferences)
- JSON dışa aktar / içe aktar (şimdilik log/clipboard)
