# OmniaOrganizer Codex Pack

This folder contains the implementation-facing planning pack for
OmniaOrganizer.

Canonical product root in this repository:
- `apps/organizer`

Canonical product direction:
- mobile file manager / personal storage control system
- local-first
- modular
- grounded utility UX
- no sci-fi product drift
- no cloud-first drift
- no PDF-first drift

## How to use the pack
- Lock product identity before coding.
- Confirm the active phase before adding any feature.
- Treat `Home / Browse / Search / Storage` as the canonical Phase 1
  navigation model unless a later doc explicitly changes it.
- Treat cloud, advanced document tooling, OCR, and AI as later layers.
- Prefer controlled rewrite of old task/note assumptions over repo-wide
  reset.
