## Canonical Rules

- `wiki/` is the active strategy and context layer
- `operations/` is the active build and operator layer
- `reference/` is historical context
- `version.json` is the current build manifest

If a document cannot answer whether it is current or historical, fix that ambiguity instead of making people guess.
