# OmniaPrompt Docs Hub

This folder is the developer-facing documentation entrypoint for OmniaPrompt.

Its job is to:
- define the canonical product truth
- separate real product documents from handoff/export clutter
- give future sessions one place to start

## Current product role

- Portfolio role: incubation / secondary review
- Product posture: premium prompt and idea management product
- Current implementation shape: web app + product-local packages + Supabase foundation
