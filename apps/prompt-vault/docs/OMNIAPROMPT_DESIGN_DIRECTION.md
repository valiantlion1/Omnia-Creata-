# OmniaPrompt Design Direction

Status: logo-aligned reset pass, 2026-04-25

## Product Job

OmniaPrompt is a mobile-first prompt workspace. The first screen must help the user capture a prompt, idea, system instruction, or agent flow before it disappears. Organization comes second.

## Visual Direction

- Warm paper, graphite text, restrained OmniaCreata gold accent, muted sage as secondary support.
- The real logo source is `design/brand/omnia-creata-logo-transparent.png`; do not redraw, crop, or simplify it into an O/droplet icon.
- Content-first app surface, not a marketing hero and not a dashboard.
- Capture composer is the primary object.
- Recent work and projects are context, not KPI reporting.
- Motion is quiet: rise, focus, state change. No glow theater.

## Anti-Goals

- No black/gold vault luxury styling; gold is a small brand/action accent only.
- No generated logo fragments, fake icon derivations, or O/droplet hallucinations.
- No fake metrics as the main home layout.
- No glassmorphism dashboard shells.
- No over-branded copy.
- No visible previous-name residue on product surfaces.
