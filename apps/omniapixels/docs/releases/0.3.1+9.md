# OmniaPixels 0.3.1+9

Date: 2026-04-26

## Status

Diagnostics and performance evidence release-test build.

## Changed

- Added in-app diagnostics for recent gallery, render, upscale, export, save, and share operations.
- Added rolling frame timing capture with average frame, worst frame, >16ms, and >33ms counters in Settings.
- Added latest error capture so phone-side failures can be reported from inside the APK instead of relying only on visual observation.
- Save and share now run export rendering inside the visible busy state instead of doing heavy work before feedback appears.
- Widget coverage now checks the diagnostics surface.

## Still needs device proof

- Real Poco X6 Pro pass using the Settings diagnostics values after gallery load, slider drag, crop render, export, and share.
- Follow-up fixes should be driven by the slowest recorded operations and frame counters, not by guesswork.
