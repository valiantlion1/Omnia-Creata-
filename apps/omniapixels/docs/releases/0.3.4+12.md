# OmniaPixels 0.3.4+12

Date: 2026-04-27

## Status

Editor hardening release-test build.

## Research basis

- Lightroom mobile keeps the photo in loupe view and groups edits into panels such as Actions, Presets, Crop, Edit, Masking, and Remove.
- Apple Photos keeps Adjust, Filters, and Crop close to the image, with swipeable settings and direct before/after compare.
- Snapseed treats tools and filters as a non-destructive stack, with quick hold-to-compare behavior.
- Material guidance supports continuous sliders for subjective intensity controls such as brightness and color saturation.

## Changed

- `Detail` now affects the live preview matrix, so sharpening/detail presets and slider movement produce immediate visual feedback while export still uses the higher-quality render path.
- Editor quick actions now include accessible Undo, Redo, and Reset all controls with tooltips.
- Presets now show active selection when their adjustment values match the current edit.
- Tool rail buttons now show a small active-change marker when that tool has non-default edits.
- Slider value labels now show meaningful neutral-relative values instead of only normalized 0-100 positions.
- Version bumped to `0.3.4+12` for release-test APK discipline.
