# OmniaPixels 0.2.1+3

Date: 2026-04-26

## Status

Gallery and performance hotfix after real phone feedback.

## Fixed

- Removed automatic first-photo selection during gallery load.
- Removed full-resolution decode during normal gallery open.
- Added album chips from the device photo library.
- Added paged loading with a `More` tile instead of trying to reason over the whole library at once.
- Added Android system picker fallback through `image_picker` for cases where full album permission behaves badly.
- Live edit now uses preview-sized image bytes instead of original-size bytes.
- Export still tries to use original bytes when saving, but only at export time.

## Still needs device proof

- Behavior on very large 25k+ photo libraries.
- Android permission behavior across Android 13/14/15.
- Long edit sessions on mid-range devices.
- Neural upscale model selection.
