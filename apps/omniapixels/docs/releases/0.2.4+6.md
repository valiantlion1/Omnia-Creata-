# OmniaPixels 0.2.4+6

Date: 2026-04-26

## Status

Editor workflow hardening.

## Changed

- Tool taps now open the tool panel instead of accidentally changing the image.
- Undo and redo controls were added to the edit surface.
- Slider edits are grouped as one undo step per drag instead of one step per frame.
- Before/after hold comparison was added directly on the image preview.
- Crop now supports `Original`, `1:1`, `4:5`, and `16:9`.
- Crop/export rendering now bakes the selected crop ratio.
- Small edit actions now use tighter touch feedback.

## Still needs device proof

- Physical-device frame timing on Poco X6 Pro.
- Manual QA for undo/redo with crop, rotate, light, color, detail, and FX.
