# OmniaPixels 0.3.2+10

Date: 2026-04-26

## Status

Gallery and editor performance hardening release-test build.

## Changed

- Gallery thumbnails now use smaller `ThumbnailOption` requests matched to the grid preview size.
- Native thumbnail pre-caching is requested for the visible and next gallery window.
- Thumbnail futures are capped with a lightweight LRU limit to avoid unbounded memory growth while scrolling large libraries.
- Gallery now auto-loads the next page near the bottom while preventing duplicate load-more requests.
- Editor preview loading now uses a lighter preview source and bounded decode width while keeping export rendering on the higher-quality path.
- Settings diagnostics now expose loaded photo count, loaded page count, and thumbnail cache usage.
- Release APK was clean rebuilt and emulator-smoked; Settings shows the visible `0.3.2+10` build label.
