# OmniaPixels 0.2.0+2

Date: 2026-04-26

## Status

First functional in-app processing seed.

## Included

- Android photo library permissions
- Real gallery loading with device thumbnails
- Photo selection from the gallery grid
- Local preview rendering off the UI isolate
- Working crop-to-square toggle
- Working 90-degree rotate action
- Working exposure, contrast, warmth, saturation, and detail controls
- On-device 2x upscale baseline using high-quality cubic resize plus detail sharpening
- Export as a new JPG copy through Android MediaStore
- Share sheet export
- Processing caps to reduce crash risk on large phone images

## Still not included

- Neural/open-source TFLite upscale model
- Manual crop box UI
- PNG/HEIC export
- Batch processing
- Release signing for Play Store

## Notes

The 2x upscale is an honest local baseline, not a neural super-resolution model yet. Neural model integration should be evaluated separately for APK size, device memory, inference speed, and Play Store stability.
