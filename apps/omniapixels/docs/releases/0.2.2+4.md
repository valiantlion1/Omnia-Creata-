# OmniaPixels 0.2.2+4

Date: 2026-04-26

## Status

Performance hotfix for large Android galleries and live editing.

## Changed

- `Choose Photos` now opens the Android/system photo picker instead of loading the custom gallery first.
- Custom gallery remains available through `Load Albums`, but it is no longer the primary path.
- Custom gallery now uses lighter first pages and cached thumbnail futures to avoid repeated media-store thumbnail work while scrolling.
- Live edit sliders no longer re-decode or re-render the image on every movement.
- Exposure, contrast, warmth, and saturation previews now use a Flutter `ColorFilter.matrix` paint path.
- CPU image rendering now happens for geometry changes, upscale, save, and share, not every slider frame.
- Release-optimized APK is now the preferred phone test artifact; debug APK is not representative for FPS.

## Still needs device proof

- Poco X6 Pro large library test with 25k+ images.
- Profile/release FPS measurement.
- Native embedded picker exploration after the app shell is stable.
