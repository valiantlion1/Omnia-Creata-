# OmniaPixels 0.3.6+14

Date: 2026-04-27

## Theme

Market-researched creator tool slice.

## What changed

- Researched Lightroom mobile, Snapseed, Picsart, Photoshop Express, Android photo picker, and Flutter performance guidance for the next editor direction.
- Added `docs/EDITOR_MARKET_RESEARCH.md` as the product reference note for editor depth.
- Added Brush, Text, and Sticker tools to the editor rail.
- Added live markup preview over the photo.
- Added undo/redo snapshots that include both edit values and markup layers.
- Added export/upscale bake support so brush, text, and sticker layers are written into saved output instead of remaining UI-only.
- Bumped Flutter app version to `0.3.6+14`.

## Product truth

- Brush/Text/Sticker are the first real creator-layer slice.
- GIF import, sticker library, object remove, local AI masking, text drag/scale/rotate, and brush eraser are not claimed in this build.
- Upscale remains local `Fast 2x` / `Enhanced 2x`; no hidden cloud model and no fake AI label.
