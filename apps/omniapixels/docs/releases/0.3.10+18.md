## What changed

- Rebuilt the visual system around a darker, calmer, image-first mobile editor surface.
- Replaced the bulky Material bottom navigation with a compact OmniaPixels bottom rail.
- Reworked the first-run screen with the real Omnia Creata logo, an editor-style preview composition, shorter copy, and cleaner actions.
- Reworked the editor empty state into a full preview canvas instead of a centered empty message with dead controls.
- Removed disabled editor control clutter when no photo is loaded.
- Reworked gallery empty state, thumbnail rhythm, preview frame, compare preview, settings header, settings rows, diagnostics panel, chips, buttons, and compact action controls.
- Replaced decorative blob texture with a quieter grid/guide texture.
- Added tooltips/content descriptions to icon buttons seen in emulator UI hierarchy.
- Updated the app version to `0.3.10+18`.

## Product truth

- This is a visual/product-quality pass, not a new AI/model release.
- The app still needs real-device visual review on a large gallery and on lower-memory Android phones.
- The next editor depth pass should focus on direct crop handles/straighten and movable text/sticker layers.
