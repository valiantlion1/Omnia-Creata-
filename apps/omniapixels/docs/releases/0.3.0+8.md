# OmniaPixels 0.3.0+8

Date: 2026-04-26

## Status

Premium editor polish plus controlled AI upscale prototype.

## Changed

- Edit model and image renderer were moved out of `main.dart` into `lib/editor/`.
- Upscale now has an `UpscaleEngine` with `Fast` and `AI Experimental` modes.
- `AI Experimental` checks for a bundled `assets/models/esrgan_2x.tflite` model and falls back safely when it is not bundled yet.
- Upscale UI now exposes the mode choice without claiming final AI quality.
- Presets, tool reset, crop guides, and improved editor feedback are included in the same release-test build.

## Still needs device proof

- Poco X6 Pro FPS and touch-latency pass.
- Real TFLite model asset selection, license review, and device proof before claiming true AI quality.
