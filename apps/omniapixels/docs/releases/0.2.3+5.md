# OmniaPixels 0.2.3+5

Date: 2026-04-26

## Status

Editor interaction polish for faster, denser, more visual controls.

## Changed

- Edit tools are now smaller and horizontally scrollable.
- Tool selection has tighter animated feedback.
- Each tool opens a compact control panel underneath it.
- Light now includes exposure, brightness, and contrast.
- Color now includes saturation, warmth, and tint.
- Detail now includes detail and clarity.
- FX adds fade and vignette with live preview on the image.
- Export/render path now bakes the new brightness, tint, clarity, fade, and vignette controls.
