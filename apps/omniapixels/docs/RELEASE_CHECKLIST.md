# OmniaPixels Release Checklist

## Before Play Store internal test

- `flutter analyze`
- `flutter test`
- Android debug smoke on a device or emulator
- Android release build
- Version code incremented
- App name and package id verified
- Omnia Creata logo renders from the real asset
- Photo permission copy is honest
- Export does not overwrite originals
- Privacy note covers local processing and any data leaving the device

## Before public launch

- Store listing screenshots from the real app
- Short and full descriptions written from product truth
- Privacy policy linked
- Crash-free smoke on at least one low/mid Android device
- No placeholder, beta, backend, roadmap, or fake AI-platform copy in product UI
