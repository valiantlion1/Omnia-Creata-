# OmniaPixels

OmniaPixels is being rebuilt from zero as a local-first mobile photo editor and upscaler by Omnia Creata.

## Current direction

- Product surface: Flutter mobile app
- Package id: `com.omniacreata.omniapixels`
- Current development version: `0.3.6+14`
- Launch target: Play Store only after the edit, upscale, export, versioning, privacy, and release checks are stable

## Product promise

Pick a photo, improve it on-device, compare the result, and save or share a clean copy. OmniaPixels is not a cloud image-generation dashboard and should not inherit old backend-first assumptions.
